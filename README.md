# products-kafka-producer

A Kafka producer that streams product data.

## Usage

1.  Clone the repository.
2.  Create a `.env` file based on `.env.example` and fill in your Kafka credentials.
3.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
4.  Run the producer:
    ```bash
    python product_producer.py
    ```
